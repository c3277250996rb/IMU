# ①. `quaternion.c`中赋值数组的变量说明

|   | 变量           | 解释                                                                                           |
| - | -------------- | ---------------------------------------------------------------------------------------------- |
| 1 | Q_Record_Flag  | 判断是否开始, 继续或者结束采集的 `标志位`                                                    |
| 2 | Q_Record_Index | 当前记录的 `索引`                                                                            |
