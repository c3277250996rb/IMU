新版ukf_大纲

最优估计: Xukf ;  
输入变量个数(维度): L

把 n-1步 的Xukf生成sigma点集;
①: Xukf_n-1  ->  sigma  ->  Xsigma_1(2L+1)

把 n-1步 的sigma通过gfun转移预测成 n步 的 预测距离;
②: Xsigma_1(2L+1)  ->  gfun(状态转移)  ->  Xsigma_1_predict(2L+1)

生成均值和协方差
③: Xsigma_1_predict(2L+1)  ->  Xs_1_p_weight(均值) , Xs_1_p_covariation(协方差)

把 n步 的均值 生成sigma点集
④: Xs_1_p_weight(均值)  ->  sigma  ->  Xsigma_UT(2L+1)

把 n步 的sigma通过hfun观测映射成 n步 的 观测距离;
⑤: Xsigma_UT(2L+1)  ->  hfun(观测映射)  ->  Zsigma_UT(2L+1)

生成均值和协方差
⑥: Zsigma_UT(2L+1)  ->  Zs_UT_weight(均值) , Zs_UT_covariation(协方差)

更新卡尔曼增益
⑦: XZs_UT_covariation =  XZs_UT_covariation + ...
                                           Wc(k) * ...
                                           (Xsigma_UT - Xs_1_p_weight) * ...
                                           (Zsigma_UT - Zs_UT_weight )';

      K = XZs_UT_covariation * Zs_UT_covariation^-1;


更新出 n步 的Xukf
⑧: Xukf_n = Xs_1_p_weight + K * (accelerate(t) - Zs_UT_weight);
