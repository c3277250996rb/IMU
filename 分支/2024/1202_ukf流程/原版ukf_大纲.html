原版ukf_大纲

最优估计: Xukf ;  
输入变量个数(维度): L

把 n-1步 的Xukf生成sigma点集;
①: Xukf_n-1  ->  sigma  ->  Xsigma(2L+1)

把 n-1步 的sigma通过gfun转移预测成 n步 的 预测距离;
②: Xaugsigma(2L+1)  ->  gfun(状态转移)  ->  Xsigmapre(2L+1)

生成均值和协方差
③: Xsigmapre(2L+1)  ->  Xpred(均值) , Ppred(协方差)

把 n步 的均值 生成sigma点集
④: Xpred(均值)  ->  sigma  ->  Xaugsigma(2L+1)

把 n步 的sigma通过hfun观测映射成 n步 的 观测距离;
⑤: Xaugsigma(2L+1)  ->  hfun(观测映射)  ->  Zsigmapre(2L+1)

生成均值和协方差
⑥: Zsigmapre(2L+1)  ->  Zpred(均值) , Pzz(协方差)

更新卡尔曼增益
⑦: Pxz=Pxz+Wc*(Xaugsigma-Xpred)*(Zsigmapre-Zpred)';
      K=Pxz*(Pzz)^(-1);

更新出 n步 的Xukf
⑧: Xukf_n  =  Xpred+K*(Z(n)-Zpred);
